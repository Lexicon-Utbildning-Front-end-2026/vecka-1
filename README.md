## 📅 Onsdag: WCAG Intro + Semantisk HTML

På eftermiddagen gäller eget arbete där ni först och främst tittar på den e-learning som finns för dagen. Komplettera med läsning utifrån vad ni tycker känns relevant från länkarna nedan. 

Målet utöver det denna vecka är att förbättra er receptsida med bättre semantik och tillgänglighet, ni kommer få visa upp detta under nästkommande vecka. På fredag kommer vi jobba med lite olika typer av tester för att se hur tillgängliga sidorna är. 

### 🎯 Mål för dagen

* Förstå vad WCAG är
* Känna till WCAG och principer för tillgänglighet
* Förstå semantisk HTML och varför det är viktigt
* Kunna anpassa HTML för god tillgänglighet (landmarks, headings, list, images, buttons, links osv)

## 📅 Fredag: Testning

Olika tester av kontraster mm, med WAVE, screenreader, tab/tangentbordsnavigation och ev andra verktyg. Skapa skip links

### 🎯 Mål för dagen

* Kunna testa tillgänglighet med hjälp av verktyg som WAVE och screenreaders
* Kunna använda olika verktyg som WAVE, tangentbord och skärmläsare för att testa tillgänglighet
